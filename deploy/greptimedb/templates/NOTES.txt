Example routines to create and connect a GreptimeDB or GreptimeDB-Cluster using kbcli.

1. Create a GreptimeDB or GreptimeDB-Cluster

Minimal >
kbcli cluster create my-greptimedb --cluster-definition greptimedb

Replicas >
helm install my-greptimedb-cluster ./deploy/greptimedb-cluster

2. Port-forward svc to localhost and connect to GreptimeDB

MySQL >
kubectl port-forward svc/my-greptimedb-frontend 4002:4002
mysql -h 127.0.0.1 -P 4002

PostgreSQL >
kubectl port-forward svc/my-greptimedb-frontend 4003:4003
psql -h 127.0.0.1 -p 4003
