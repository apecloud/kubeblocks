# 创建数据库集群

KubeBlocks 通过插件（Addon）机制支持多种数据库引擎。部署 KubeBlocks 后，您可以使用 KubeBlocks 创建各种数据库集群，例如 MySQL、PostgreSQL、Redis、MongoDB、Kafka、Pulsar、RabbitMQ、Elasticsearch、Qdrant 等。具体创建步骤可参考相应的文档。
