# 升级到 KubeBlocks v0.7

**步骤：**

1. [安装 kbcli](../../installation/install-with-kbcli/install-kbcli.md) 并确保其版本为 v0.7.0。

2. 将 KubeBlocks 从 v0.6 升级到 v0.7。

   ```bash
   kbcli kubeblocks upgrade --version=0.7.0
   ```
